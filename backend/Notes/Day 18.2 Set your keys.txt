OPEN PowerShell (Run once)

setx AWS_ACCESS_KEY "YOUR_ACCESS_KEY_ID"
setx AWS_SECRET_KEY "YOUR_SECRET_ACCESS_KEY"

setx RAZORPAY_KEY_ID "Your Key"
setx RAZORPAY_KEY_SECRET "Your Key"


echo $env:AWS_ACCESS_KEY
echo $env:AWS_SECRET_KEY

echo $env:RAZORPAY_KEY_ID
echo $env:RAZORPAY_KEY_SECRET

========================================================================================================

Keep in mind that you have to do this because in application.yml file you cannot write directly ACCESS KEY and SECRET KEY for that you have to do this otherwise while pushing code in githud or any other git platform it will not allow you to upload your code. For that you have to set in you system environment you can achieve that by the above comment.
